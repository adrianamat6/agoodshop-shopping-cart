# agoodshop-shopping-cart

Este proyecto es una implementación de un carrito de la compra interactivo para una tienda ficticia.

## Objetivos
- Consumo de datos desde una API externa (JSONBlob).
- Manipulación dinámica del DOM sin refrescar la página.
- Uso de Programación Orientada a Objetos (Clase `Carrito`).

## Tecnologías
- HTML5 & CSS3 (Maquetación)
- JavaScript (Vanilla ES6+)
